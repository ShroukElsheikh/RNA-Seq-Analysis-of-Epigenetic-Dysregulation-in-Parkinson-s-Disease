# RNA-Seq-Analysis-of-Epigenetic-Dysregulation-in-Parkinson-s-Disease
Performing end-to-end RNA-Seq analysis including raw read QC (FastQC/Trimmomatic), pseudo-alignment (Kallisto/GRCm39), and differential expression analysis using both DESeq2 and Limma pipelines on a publicly available Parkinson's disease dataset (GSE180408)
